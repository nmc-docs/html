# Các thẻ liên quan đến text

| Tên thẻ   | Ý nghĩa                         |
| --------- | ------------------------------- |
| p         | Định nghĩa một đoạn văn bản     |
| h1 ... h6 | Định nghĩa tiêu đề cho đoạn văn |
| em        | Chữ in nghiêng                  |
| del       | Chữ bị gạch ở giữa              |
| sup       | Chữ nhỏ bên trên (giống số mũ)  |
| sub       | Chữ nhỏ bên dưới                |
| br        | Xuống dòng                      |
